README

The 'recipes' contained herein are samples how to establish different 'toolchains' in MSYS2.

Focus is on using UI libraries; however, some toolchains do not include these, but samples are given in the 'samples' folder.

#For GTK, there are two recipes targetting the UCRT64 toolchain. One is GTK3, the other GTK4. These recipes follow the path outlined on the GTK website, .s. https://www.gtk.org/docs/language-bindings/.

#An other recipe has been added to show GTK development with Rust on the CLANG64 toolchein. 

#For Qt, there are also two recipes, one targetting Qt5 and the other Qt6. Both use the CLANGARM64 toolchain. 

#The 'mingw64-qt6-python3' follows the recommended way to develop Qt6 applications on Windows.
